Scratchie Favicon PNG Generator

This page helps generate PNG versions of the Scratchie carrot favicon for browsers that don't support SVG favicons.

Instructions:

  1. Click each "Download" button to save the PNG file
  2. Save the files with the exact names shown
  3. Place them in the /public directory
  4. For the .ico file, you'll need to use an online converter or tool to combine the PNGs
